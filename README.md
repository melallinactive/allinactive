# ALLINACTIVE

A Python Flask web application for showcasing activewear brands with user signup tiers.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python main.py
   ```

3. Open your browser to `http://127.0.0.1:8000/allinactive`

## Features

- Frontend page with title "ALLINACTIVE"
- Placeholders for activewear brands
- Popup modal for new users to sign up to Tier 1, Tier 2, or Tier 3
